# LogProxyApi

## How to build and run
1. Clone the repository
2. Navigate to the ```{REPO-PATH}\src\LogProxyApi``` and execute ```dotnet build```
3. Add the AirTable API Key as Environment Variable with the Key ```AIR_TABLE_API_KEY```
4. Run the project with ```dotnet run```

## Trigger endpoints with postman
To load the endpoints into postman you can import the file in the ```postman``` folder.

## Basic Authentication
You can use any user-password-combination defined in the ```{REPO-PATH}\src\LogProxyApi\appsettings.json``` file.
